## 3. 实现要点

### 3.1 Alignment Chunk

- **Chunk 构建**：按 `chunk_size`（默认 100）将 tasks 打成块，每块 `(indices, paths_chunk, seed_rows_chunk)`。
- **Worker**：`_alignment_chunk_worker_impl` 循环处理块内颗粒，局部累加 `local_acc += tr`，返回 `(rows_list, local_acc)`。
- **主进程**：`avg_acc += local_acc`，`rows_pairs.extend(rows_list)`；chunk 模式下 tbl 流式改为收集后排序再写。
- **Fallback**：`chunk_size=1` 时使用原有单颗粒 worker `_alignment_cpu_worker_impl`。

### 3.2 Classification Initializer + Chunk

- **Initializer**：`_classification_worker_init(ref_vols, align_mask, wedge_mask, align_params)` 注入 worker 全局变量。
- **Chunk worker**：`_classification_chunk_worker_impl` 处理多颗粒，返回 `(rows_list, per_ref_local_acc)`；主进程合并 `per_ref_acc[r] += per_ref_local_acc[r]`。
- **Single-particle**：`_classification_cpu_worker_impl` 使用 globals，不再从磁盘加载 ref。
