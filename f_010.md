## Required modifications / 必须修改

### 1) Add missing GPU regression coverage (must-fix)
### 1) 补齐 GPU 回归覆盖（必须）

**Problem / 问题**
- Current tests proving multigrid+wedge shape safety are CPU-only.
- 目前证明 multigrid+wedge 形状安全的用例仅覆盖 CPU。

**Required change / 必须修改**
- Add GPU-targeted regression tests in `pydynamo/test/test_align.py`:
  1. `multigrid_levels=2` + direct `wedge_mask` + `device="cuda"` should run without shape/broadcast mismatch.
  2. `multigrid_levels=2` + `fsampling_mode="table"` + `wedge_apply_to="auto"` + `device="cuda"` should run without shape/broadcast mismatch.
- 在 `pydynamo/test/test_align.py` 新增 GPU 定向回归测试：
  1. `multigrid_levels=2` + 直接 `wedge_mask` + `device="cuda"`，不得出现 shape/broadcast 错误；
  2. `multigrid_levels=2` + `fsampling_mode="table"` + `wedge_apply_to="auto"` + `device="cuda"`，不得出现 shape/broadcast 错误。

**Acceptance / 验收标准**
- On CUDA-available machine, both tests pass and no broadcast-shape warning/error occurs.
- 在可用 CUDA 机器上，两条测试通过，且不出现 broadcast-shape 警告/错误。

---

### 2) Replace risky wedge resample fallback behavior
### 2) 替换有风险的 wedge 重采样兜底行为

**Problem / 问题**
- `_resample_wedge_mask_to_shape(...)` currently falls back to `np.resize` when shape still mismatches after zoom.
- 当前 `_resample_wedge_mask_to_shape(...)` 在 `zoom` 后形状仍不匹配时使用 `np.resize` 兜底。

**Risk / 风险**
- `np.resize` may repeat/truncate data and is not a geometric resampling operation; this can introduce non-physical wedge mask artifacts.
- `np.resize` 可能重复/截断数据，不是几何意义上的重采样，可能引入非物理 wedge mask 伪影。

**Required change / 必须修改**
- Replace `np.resize` fallback with deterministic geometric handling:
  - either exact coordinate resampling to target grid,
  - or explicit center-aligned crop/pad after interpolation.
- 将 `np.resize` 替换为确定性的几何处理：
  - 要么按目标网格坐标精确重采样；
  - 要么插值后做显式中心对齐 crop/pad。

**Acceptance / 验收标准**
- Add unit test covering non-integer scale transitions (e.g., odd/even and non-2x mapping), verifying:
  - output shape exactly equals target shape;
  - values remain bounded in `[0, 1]`;
  - no data repetition artifact pattern from `np.resize`.
- 新增单测覆盖非整数比例变化（如奇偶尺寸、非 2x 映射），并验证：
  - 输出形状严格等于目标形状；
  - 数值保持在 `[0, 1]`；
  - 不存在 `np.resize` 式重复伪影。
