### 1.2 chunk_size=1 等价性回归测试

#### 1.2.1 Alignment 测试

| 项目 | 内容 |
|------|------|
| 文件 | `pydynamo/test/test_alignment_command.py` |
| 测试名 | `test_alignment_chunk_size_equivalence` |
| 步骤 | 1. 创建 synthetic 数据（ref + 多颗粒，如 4 个）<br>2. 以 `num_workers=2`、`chunk_size=1` 运行 alignment<br>3. 以 `num_workers=2`、`chunk_size=100` 运行 alignment<br>4. 断言 tbl/star 关键列（tdrot, tilt, narot, dx, dy, dz, cc）在浮点误差内一致<br>5. 断言 output average.mrc 在 `np.allclose(..., atol=1e-4, rtol=1e-3)` 内一致 |
| 输出 | output_table 为 .star 或 .tbl；output_average |

#### 1.2.2 Classification 测试

| 项目 | 内容 |
|------|------|
| 文件 | `pydynamo/test/test_classification.py` |
| 测试名 | `test_classification_chunk_size_equivalence` |
| 步骤 | 1. 创建 synthetic 数据（单 ref + 多颗粒，如 4 个）<br>2. 单迭代、swap=True，以 `num_workers=2`、`chunk_size=1` 运行<br>3. 以 `num_workers=2`、`chunk_size=100` 运行<br>4. 断言 per-ref 输出行数一致<br>5. 断言 average 体积在合理误差内一致 |
| 输出 | ite_001/refined_table_ref_001.tbl、ite_001/average_ref_001.mrc |
